# Linux Notes

1. Echo: Prints the text on terminal
 ``` 
 echo
```

2. Whoami : Returns the current user
```
whoami
```

3. id: Returns the user ID, group ID and groups to which the user belongs

4. id -un

5. ls - list directories
    ls - a viewing hidden files
    ls - r view in reverse order
    ls - l getting detailed info
    ls - la comining both of the above command

6. touch - create a new file

7. pwd - Print Working Director

8. cd - change directory

9. file - tell me the file type

10. cat - (concatinate) use to view, create and combine file
     cat > file.txt - create a file
     cat >> file.txt -  append content to the file
     cat file1.txt file2.txt > merged txt - combine multiple files
     cat file1.txt file2.txt view multiple files
